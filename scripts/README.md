# Scripts Directory

This directory contains automation and deployment scripts.

## Available Scripts

- `export_model.py` - Export trained ML models to ONNX format
- `deploy.sh` - Deployment automation script
- `benchmark.sh` - Performance benchmarking for DSP code
- `test_audio.sh` - Run JUCE unit tests

## Guidelines

- Keep scripts idempotent
- Use clear error messages
- Document required dependencies
- Include usage examples in script headers
